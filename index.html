Orbits

A set of scattered random points are each connected to another point at random. Each point then moves towards its target at a speed inversely proportional to its distance to the target. The lines between each object and its target is traced frame by frame, creating the mesh like structure.

Strange Attractors

The deJong attractor is a mathematical system that evolves in time, parameterized by four parameters \( a, b, c \) and \( d \). The system is determined by the following equations: $$ x_{t+1} = \sin(a y_t) - \cos(b x_t), \\ y_{t+1} = \sin(c x_t) - \cos(d y_t). $$ The parameters are selected randomly in the interval \( [-3, 3] \).

Boids

A set of boids are attracted to the center of mass of the other boids, while also trying to keep a certain distance from the other boids. This creates a flocking effect as seen in schools of fish and flocks of birds.

Celestial Attraction

A set of scattered points are each assigned a target, and a target radius. Each point then attempts to move into orbit around the assigned target maintaining this fixed radius. The paths are traced frame by frame. About 50% of the time, all objects are assigned the same target.

Splitter

A circle is split, yielding a random number of new circles in uniformly distributed angles from the center of the original circle. This process is then recursively applied.
